使用Git的问题合集

张启帆 / 2022-10-11

我将在这里记下使用git的时的问题。

1 Errno 10054 #

fatal: unable to access ‘https://github.com/…': OpenSSL SSL_read: Connection was reset, errno 10054 原因是因为服务器的SSL证书没有经过第三方机构的签署1，解决方法为解除SSL验证： git config –global http.sslVerify false
